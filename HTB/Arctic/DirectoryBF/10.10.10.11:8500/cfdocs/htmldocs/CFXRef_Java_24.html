write

Description

Outputs text back to the user.

Category

Response interface

Syntax

public void write(String output)

Parameters

Parameter

Description
output

Text to output

Example

The following example outputs the value of the DESTINATION attribute: 

response.write( "DESTINATION = " + 
    request.getAttribute("DESTINATION") ) ;
